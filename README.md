## 📖 About the Project

**AtmosX** is a lightweight and user-friendly weather forecasting web application that allows users to search for any city and instantly view its current weather conditions.

The application focuses on delivering an intuitive user experience with responsive design, smooth interactions, and accurate weather information.

Whether you're planning your day or simply checking the current weather, AtmosX provides essential weather details in an elegant interface.

---

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature information
- ☁️ Weather condition display
- 💨 Wind speed details
- 💧 Humidity information
- 📍 Dynamic location updates
- 📱 Fully Responsive Design
- ⚡ Fast and Lightweight
- 🎨 Modern UI/UX Design

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/0XRC-1973/AtmosX.git
```

### Navigate to the Project

```bash
cd AtmosX
```

### Run the Application

Simply open the `index.html` file in your preferred web browser.

No installation is required.
