# 📖 About

**AtmosX** is a modern weather forecasting web application that allows users to search for any city and instantly receive accurate weather information in a clean and responsive interface.

Designed with simplicity and performance in mind, AtmosX delivers real-time weather updates, essential atmospheric information, and an intuitive user experience across desktop and mobile devices.

---

# ✨ Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature updates
- ☁️ Weather condition display
- 💨 Wind speed information
- 💧 Humidity details
- 📍 Dynamic location search
- ⚡ Fast & lightweight
- 📱 Fully responsive design
- 🎨 Modern UI/UX
- 🔄 Instant weather updates

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/0XRC-1973/AtmosX.git
```

## Navigate into the project

```bash
cd AtmosX
```

## Run the Application

Simply open the **index.html** file in your web browser.

No additional installation or dependencies are required.
